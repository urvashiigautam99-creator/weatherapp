# 🌦 Weather App

A simple weather application built using HTML, CSS, and JavaScript.

## 🚀 Live Demo
https://weatherapp-eight-iota.vercel.app/

## 📌 Features
- Search weather by city
- Shows temperature
- Weather condition display
- Responsive UI

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API
